# tpch_dbt

## Case Scenario
This project is a case study focused on applying analytics engineering
principles using dbt Cloud and Snowflake.

## Data Source
The project uses TPC-H SF1 sample data available in Snowflake.

- Database: SNOWFLAKE_SAMPLE_DATA
- Schema: TPCH_SF1

## Tools & Technologies
- Snowflake
- dbt Cloud
- GitHub

## Objective
To build a well-structured dbt project that transforms TPC-H SF10 data
using staging and marts models, following best practices.
